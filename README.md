# Álgebra y Geometría Analítica — Parcial Teórico

Sitio interactivo de repaso para AGA UTN.

## Contenido

- Vectores
- Combinación lineal
- Independencia y dependencia lineal
- Matrices
- Rango
- Determinantes
- Sistemas lineales
- Rouché-Frobenius
- Preguntas típicas de parcial
- Modo repaso con flashcards

## GitHub Pages

Para publicarlo:

1. Ir a `Settings`.
2. Entrar a `Pages`.
3. En `Build and deployment`, elegir `Deploy from a branch`.
4. Branch: `main`.
5. Carpeta: `/root`.
6. Guardar.

La página quedará disponible en:

`https://gr4vity69.github.io/algebra_y_geometria/`
