1. Vectores

Qué es un vector

Un vector representa una magnitud con módulo, dirección y sentido. En R² se escribe como v = (x, y); en R³ como v = (x, y, z).

v = (3,2)

Módulo

Es la longitud del vector. En R²:

|v| = √(x² + y²)

Ejemplo: si v=(3,4), entonces |v|=5.

Paralelismo

Dos vectores son paralelos si sus componentes son proporcionales. Ejemplo:

(2,4) = 2(1,2)

Entonces son dependientes lineales.

2. Combinación lineal, LI y LD

Combinación lineal

Un vector es combinación lineal de otros si puede construirse usando esos vectores multiplicados por escalares.

v = λ₁v₁ + λ₂v₂ + ... + λₙvₙ

Idea humana: ¿puedo fabricar este vector usando los otros?

Independencia lineal

Un conjunto es LI si la única forma de obtener el vector nulo es usando todos los escalares iguales a cero.

Traducción: ningún vector sobra.

Interpretación geométrica

En R², dos vectores LI no son paralelos y pueden generar todo el plano. Dos vectores LD son paralelos. En R³, tres vectores LD pueden estar en un mismo plano.

3. Matrices

Definición

Una matriz es un arreglo rectangular de números. Su orden es filas × columnas.

Producto

El producto matricial generalmente NO es conmutativo.

AB ≠ BA

Inversa

Una matriz cuadrada A tiene inversa si existe A⁻¹ tal que:

AA⁻¹ = A⁻¹A = I

4. Rango

El rango mide cuánta información independiente tiene una matriz. Se puede pensar como la cantidad de filas o columnas linealmente independientes.

  • Rango máximo en una matriz n×n: n.
  • Si rango menor al máximo, hay dependencia lineal.
  • En sistemas, el rango sirve para saber si hay solución única, infinitas o ninguna.

5. Determinantes

Qué son

Un determinante es un número asociado a una matriz cuadrada. Sirve para estudiar invertibilidad, rango, independencia lineal y sistemas.

2×2

|a b; c d| = ad - bc

Propiedades que salvan parciales

  • Dos filas iguales → det = 0.
  • Una fila múltiplo de otra → det = 0.
  • Intercambiar filas cambia el signo.
  • Matriz triangular → producto de la diagonal.
  • det(A) ≠ 0 → A tiene inversa.

6. Sistemas de ecuaciones lineales

Compatible determinado

Tiene una única solución. Geométricamente: rectas que se cortan en un punto.

Compatible indeterminado

Tiene infinitas soluciones. Geométricamente: rectas coincidentes o planos que se superponen.

Incompatible

No tiene solución. Geométricamente: rectas paralelas o planos sin intersección común.

Rouché-Frobenius

  • Si rango(A) ≠ rango(A*) → incompatible.
  • Si rango(A) = rango(A*) = n → compatible determinado.
  • Si rango(A) = rango(A*) < n → compatible indeterminado.

Preguntas típicas de teoría

¿Qué significa det(A)=0?

A es singular, no tiene inversa, su rango es menor que el orden, sus columnas o filas son LD y el sistema asociado no tiene solución única.

¿Qué representa el rango?

La cantidad de filas o columnas independientes. En sistemas ayuda a clasificar soluciones.

¿Cuál es la diferencia entre LI y LD?

LI: ningún vector sobra. LD: algún vector puede obtenerse como combinación lineal de otros.

¿Cuándo se puede usar Cramer?

Cuando el sistema es cuadrado y det(A) ≠ 0.
